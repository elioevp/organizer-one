
dame el comado sql en COSMODB para saber los registros de username ="mary ramos" y directorio ="liquidacion-abril25"

Para ubicar los registros que cumplen con ambas condiciones (username = "mary ramos" y directorio = "liquidacion-abril25"), debes
 usar una consulta SQL con múltiples cláusulas WHERE unidas por el operador lógico AND.

Aquí tienes el comando de consulta:

SQL

SELECT *
FROM c
WHERE c.username = "mary ramos" AND c.directorio = "liquidacion-abril25"
